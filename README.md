ThreatWatch
A real-time threat intelligence dashboard that pulls live data from AbuseIPDB, AlienVault OTX, and URLhaus — three professional-grade public threat feeds — and displays it through a clean, interactive web dashboard.
You can monitor a live feed of malicious IPs and URLs as they're reported globally, filter threats by country using an interactive world map, search any IP or URL for its full threat history, and view a dedicated South Africa context panel showing local attack patterns and a documented SA cyber incident timeline.
Built with FastAPI, PostgreSQL, Leaflet.js, and Chart.js. Runs fully in Docker.
